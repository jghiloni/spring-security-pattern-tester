About this project

Spring Security uses the concepts of Request Matchers to affect changes to an application's security to only certain parts of an application. This project aims to provide a simple mechanism for testing that a pattern matches against a given URI. You can run this Spring Boot application locally or try it below.

To clone the repository for local use:

$ cd your_repo_root
$ git clone https://github.com/ECSTeam/spring-security-pattern-tester.git
